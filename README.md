# CJ Aleido Portfolio



## Files
- `index.html`
- `styles.css`
- `script.js`
- `images/`

## Upload to GitHub Pages
1. Open your GitHub repo.
2. Upload all files and the `images` folder.
3. Make sure `index.html` is in the root of the repo.
4. Go to Settings > Pages.
5. Set source to your main branch.
6. Wait for GitHub to publish.
